# RubyDung-TeaVM

* **You need Java 17 to compile, otherwise it won't work**

Play it here -> https://cucuzacu.github.io/Games/RubyDung

## Building(on Windows)

git clone https://github.com/Cucuzacu/RubyDung-TeaVM



cd RubyDung-TeaVM



setjavapath



gradlew generateJavaScript





Now, go into the out folder and host it as a web server!
